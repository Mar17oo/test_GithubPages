# Proyecto Estético

Este proyecto es una página web de ejemplo creada para practicar:

- HTML semántico
- Control de versiones con Git
- Publicación en GitHub Pages

## Características
- Estructura moderna y responsiva
- Uso de títulos, párrafos, enlaces, imágenes, listas y formularios
- Diseño visual atractivo

## Despliegue

1. Clona el repositorio:
   ```
   git clone https://github.com/Mari17oo/tu_repositorio.git
   ```
2. Abre el archivo `web_estetica/index.html` en tu navegador.
